**Step 2 — Create the videos storage bucket:**

The bucket must be created in the Supabase project. Use the Python service key to create it programmatically:

```python
# Run this as a one-off script from repo root
import os
from supabase import create_client
from dotenv import find_dotenv, load_dotenv

load_dotenv(find_dotenv())

client = create_client(os.environ["SUPABASE_URL"], os.environ["SUPABASE_SERVICE_KEY"])

# Create public bucket
try:
    result = client.storage.create_bucket(
        "videos",
        options={
            "public": True,
            "file_size_limit": 15728640,  # 15 MB
            "allowed_mime_types": ["video/mp4"]
        }
    )
    print("Bucket created:", result)
except Exception as e:
    if "already exists" in str(e).lower() or "Duplicate" in str(e):
        print("Bucket already exists — OK")
    else:
        raise
```

pipeline_runs_result = anon_client.table("pipeline_runs").select("*").execute()
