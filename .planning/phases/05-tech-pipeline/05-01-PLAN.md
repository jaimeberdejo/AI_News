  ```python

  ```
- Change `fetch_and_deduplicate()` signature to `fetch_and_deduplicate(category: str = "finance") -> list[Article]`.
- Inside the function, use `FEEDS_BY_CATEGORY.get(category, FINANCE_FEEDS)` to select feeds.
- The DB dedup query against `videos` table must also filter by `editions.category` to avoid cross-category deduplication. Change the DB dedup to join editions:
  ```python
  # Get today's edition IDs for this category
  today_editions = (
      db.table("editions")
      .select("id")
      .gte("created_at", str(date.today()))
      .eq("category", category)
      .execute()
  )
  edition_ids = [row["id"] for row in today_editions.data]
  existing = (
      db.table("videos")
      .select("source_url")
      .in_("edition_id", edition_ids)
      .execute()
  )
  today_urls = {row["source_url"] for row in existing.data if row["source_url"]}
  ```
  If `edition_ids` is empty, skip the DB dedup query entirely (return all fetched articles).
