  <name>Task 1: Create comments migration</name>
  <files>frontend/supabase/migrations/20260326000001_add_comments.sql</files>
  <action>
Create the migration file with the following in order:

1. Add comment_count column to videos (consistent with like_count from Phase 9):
   `ALTER TABLE public.videos ADD COLUMN IF NOT EXISTS comment_count integer NOT NULL DEFAULT 0;`

2. Create video_comments table:
   - id uuid PRIMARY KEY DEFAULT gen_random_uuid()
   - video_id uuid NOT NULL REFERENCES public.videos(id) ON DELETE CASCADE
   - user_id uuid NOT NULL REFERENCES public.profiles(id) ON DELETE CASCADE  [IMPORTANT: FK to profiles, NOT auth.users — enables Supabase embedded join syntax]
   - body text NOT NULL CHECK (char_length(body) <= 500 AND char_length(body) > 0)
   - created_at timestamptz DEFAULT now()

3. Create two indexes:
   - idx_video_comments_video ON video_comments(video_id, created_at) — for loading comments by video
   - idx_video_comments_user_ts ON video_comments(user_id, created_at DESC) — for rate limit query

4. Enable RLS: `ALTER TABLE public.video_comments ENABLE ROW LEVEL SECURITY;`

5. Create three RLS policies:
   - "anyone can read comments": FOR SELECT TO authenticated, anon USING (true)
   - "users can insert own comments": FOR INSERT TO authenticated WITH CHECK (auth.uid() = user_id)
   - "users can delete own comments": FOR DELETE TO authenticated USING (auth.uid() = user_id)

6. Create trigger function update_video_comment_count():
   - SECURITY DEFINER SET search_path = '' (established project security pattern)
   - IF TG_OP = 'INSERT': UPDATE videos SET comment_count = comment_count + 1 WHERE id = NEW.video_id
   - IF TG_OP = 'DELETE': UPDATE videos SET comment_count = GREATEST(comment_count - 1, 0) WHERE id = OLD.video_id
   - RETURN NULL (AFTER trigger on statement)

7. Create the trigger:
   - DROP TRIGGER IF EXISTS on_comment_change ON public.video_comments;
   - CREATE TRIGGER on_comment_change AFTER INSERT OR DELETE ON public.video_comments FOR EACH ROW EXECUTE FUNCTION public.update_video_comment_count();

See 10-RESEARCH.md Pattern 1 for the exact SQL.
