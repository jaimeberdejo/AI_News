    - State: `user` typed as `User | null | undefined` (undefined = loading, null = signed out, User = signed in). Use `undefined` as the initial value — NOT `null` — so consumers can distinguish "loading" from "confirmed guest".

    TypeScript: Import the `User` type from `@supabase/supabase-js` for the state type annotation.

    Anti-patterns to avoid:
    - Do NOT use `getSession()` — use `getUser()` for the initial load.
    - Do NOT put this in a context/store — a simple hook called in VideoFeed is enough for Phase 8 scope.
